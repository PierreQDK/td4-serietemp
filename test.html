TS vs DS : simulation de processus

Loi N(0,1/4)

Seed(123)

library(ggplot2)
set.seed(123)

n <- 200  
sigma2 <- 1/4  

epsilon <- rnorm(n, mean = 0, sd = sqrt(sigma2))  # Génération du bruit blanc


Y_DT <- numeric(n)
Y_TS <- numeric(n)

for (t in 1:n) {
  Y_DT[t] <- 0.2 * t + epsilon[t]  
  if (t == 1) {
    Y_TS[t] <- epsilon[t]  
  } else {
    Y_TS[t] <- 0.2 + Y_TS[t - 1] + epsilon[t]  
  }
}

# Création du DataFrame
data <- data.frame(
  Time = rep(1:n, 2),
  Value = c(Y_DT, Y_TS),
  Process = rep(c("DT: Yt = 0.2t + εt", "TS: Yt = 0.2 + Yt-1 + εt"), each = n)
)

# Graphique
ggplot(data, aes(x = Time, y = Value, color = Process)) +
  geom_line() +
  labs(title = "Comparaison entre un processus déterministe et un processus stochastique dans une loi N(0,1/4)",
       x = "Temps",
       y = "Valeur de la série") +
  theme_minimal()

Interprétation

Le graphique compare deux séries soumises à un même bruit N(0,1/4): un processus déterministe ( Y_t = 0.2t + t ), et un processus stochastique ( Y_t = 0.2 + Y*{t-1} + _t ).

La série déterministe suit une tendance linéaire régulière, perturbée temporairement par de faibles chocs : elle est stationnaire autour d’une tendance. En revanche, la série stochastique accumule les chocs dans le temps, ce qui entraîne des écarts de plus en plus marqués : elle est non stationnaire avec des effets de chocs permanents.
